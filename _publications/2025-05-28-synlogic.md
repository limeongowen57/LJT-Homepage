**Abstract**

We present SynLogic, a framework for synthesizing verifiable reasoning data at scale for learning logical reasoning and beyond. The key idea is to leverage symbolic logic systems to automatically generate reasoning chains with guaranteed correctness, which can then be used to train large language models.

Our approach enables scalable creation of high-quality training data for improving reasoning capabilities in LLMs, with experiments demonstrating effectiveness on logical reasoning benchmarks.
